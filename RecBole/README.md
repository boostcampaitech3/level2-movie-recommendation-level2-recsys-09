# RecBole

RecBole implementation for boostcamp MovieRec project.

## Installation

```bash
pip install recbole
```

## Getting Started

### train.py

Train RecSys model

### inference.py

Inference trained RecSys model

output: submission.csv

### make_data.py

Make Atomic file for RecBole
